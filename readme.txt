=== Carlos Gutenberg Blocks ===

A WordPress Plugin that creates a Gutenberg block that creates an elegant floating images slider with customizable animations and responsive design.

## Description

This Gutenberg block allows you to create an animated image slider with floating effects. Perfect for showcasing images in a dynamic and engaging way.

### Features

- Support for multiple images
- Customizable animation timings
- Responsive design with mobile optimization
- Automatic and manual sizing options
- Configurable aspect ratios (9:16, 1:1, 4:3, 4:5)
- Adjustable spacing between images
- Left/Right alignment options
- Smooth floating animations

## Installation

1. Download or clone this repository to your WordPress plugins directory
2. Navigate to the plugin directory and unzip the contents
3. Activate the plugin through the WordPress plugins page
4. Search the Gutenberg block as "Carlos Gutenberg Blocks - Images Slider"
5. Click "Add to Gutenberg" to add the block to your site
6. Customize the block settings as desired
